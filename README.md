# 📚 Student Records System

## 📌 Description
A Java application that manages student data and stores records using file handling techniques.

## ⚙️ Features
- Add student records
- Save records to file
- Retrieve and display stored data

## 🧠 Concepts Used
- File handling (FileReader, FileWriter)
- Data storage and retrieval
- Basic CRUD operations

## ▶️ How to Run
1. Compile:
   javac *.java
2. Run:
   java StudentRecords
